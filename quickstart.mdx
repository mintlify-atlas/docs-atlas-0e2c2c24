title: "Quickstart"
description: "Start generating cryptographically secure passwords with KeyForge in under two minutes."
---

KeyForge runs entirely in your browser. There is nothing to install, no account to create, and no data that ever leaves your device.

<Steps>
  <Step title="Open KeyForge">
    Navigate to [keyforgegd.vercel.app](https://keyforgegd.vercel.app) in any modern browser.

    The app loads instantly and generates a secure password automatically. You can use it offline after the first load — no network connection is required for generation.
  </Step>

  <Step title="Choose a generator mode">
    Use the four tabs at the top to switch between generation modes:

    <Tabs>
      <Tab title="Password">
        Full character-class control with configurable length (8–128 characters), anti-bruteforce guards, and custom character pools. Best for website logins, API keys, and service credentials.
      </Tab>
      <Tab title="Passphrase">
        Word-based passwords from a 200-word curated list. Memorable and mathematically strong. Best for master passwords and credentials you need to type or remember.
      </Tab>
      <Tab title="PIN">
        Numeric PINs from 4–20 digits with guards against sequences and common patterns. Best for device unlock codes, ATM PINs, and numeric access codes.
      </Tab>
      <Tab title="Phonetic Shield">
        Pronounceable syllable-based passwords with entropy injected between syllables. You can say them aloud and remember them, yet they resist all standard attack methods. Best for credentials that must be dictated or typed from memory.
      </Tab>
    </Tabs>
  </Step>

  <Step title="Configure your settings">
    Each mode has its own settings panel below the output field.

    **For Password mode:**

    | Setting | Default | What it does |
    |---------|---------|--------------|
    | Length | 16 | Drag the slider from 8 to 128 |
    | Uppercase A–Z | On | Include capital letters |
    | Lowercase a–z | On | Include lowercase letters |
    | Digits 0–9 | On | Include numbers |
    | Symbols | On | Include `!@#$%^&*` and others |
    | Exclude ambiguous | On | Remove `0`, `O`, `l`, `1`, `I` and similar |
    | Block keyboard walks | On | Prevent `qwe`, `asd`, `123` sequences |
    | Block weak fragments | On | Prevent `pass`, `admin`, `login` substrings |

    <Tip>
      Settings take effect immediately — the output regenerates live as you change any control. You do not need to click Generate after adjusting a slider or checkbox.
    </Tip>
  </Step>

  <Step title="Read the entropy output">
    Below the generated password you'll see three pieces of information:

    - **Strength label** — one of: Very Weak, Weak, Fair, Strong, Very Strong, Fortress
    - **Entropy bits** — the information-theoretic measure of unpredictability (aim for 73+ bits for real accounts)
    - **Crack time** — estimated time to brute-force at 1 trillion guesses per second

    | Strength | Bits | Crack time at 10¹² guesses/sec |
    |----------|------|-------------------------------|
    | Very Weak | < 36 | Milliseconds |
    | Weak | 36–55 | Minutes to hours |
    | Fair | 56–72 | Days to months |
    | Strong | 73–94 | Years |
    | Very Strong | 95–127 | Thousands of years |
    | Fortress | 128+ | Heat death of universe |

    The **Entropy DNA bar** above the strength meter shows each character as a colored block: coral for uppercase, silver for lowercase, cyan for digits, and gold for symbols. A varied color pattern confirms a well-mixed password.

    <Note>
      For any account that matters — email, banking, password manager — target **73 bits or higher** (Strong or better). The default 16-character password with all four character classes enabled produces approximately 104 bits.
    </Note>
  </Step>

  <Step title="Copy your password">
    You have three ways to copy:

    1. **Click the output field** — copies instantly and shows a confirmation toast
    2. **Click the ⧉ icon** on the right of the output field
    3. **Press Ctrl+Enter** (or Cmd+Enter on Mac) to regenerate, then copy

    <Tip>
      Use **Ctrl+Enter** to regenerate quickly without reaching for the mouse. The shortcut works for whichever tab is active.
    </Tip>

    After copying, paste directly into your password manager or the target field. KeyForge never stores generated passwords.
  </Step>
</Steps>

## Quick configuration reference

<Tabs>
  <Tab title="Website login">
    Password mode · Length 16 · All classes on · Ambiguous excluded · All guards on

    Produces approximately 104 bits. Suitable for any web account.
  </Tab>
  <Tab title="Master password">
    Passphrase mode · 6 words · Capitalize on · Append number on · Append symbol on

    Produces approximately 56 + 6.5 + 2.8 = 65 bits from the word selections alone plus extras. Memorable enough to type from memory.
  </Tab>
  <Tab title="PIN">
    PIN mode · 8 digits · Block sequences on · Block common patterns on

    Produces 26.6 bits — use the longest PIN your device supports for maximum security.
  </Tab>
  <Tab title="Dictated credential">
    Phonetic Shield · 4 syllable groups · Medium entropy · Capitalize on · Show guide on

    The pronunciation guide shows syllables like `BRAY · FLUND · STOO · PLAK` so you can read the password aloud to someone or type it without looking at the screen.
  </Tab>
</Tabs>
